（今天写misc）

1.1攻防世界-适合作为桌面：

​ 打开附件，得到一张图片。放到010里没发现什么，放到stedsolve里看看，一直切换，发现一张二维码，如下。

​ CQR里扫一下得到一串十六进制数字。

​ 先将数字复制粘贴到一个txt文本中，保存好打开010Editor–点击左上角‘文件’–点击‘导入十六进制’。导入完成。

​ 看到flag是.py文件，而且03 F3 0D 0A是pyc文件头。将010中的文本保存为.pyc文件，使用反编译python反编译 - 在线工具，反编译完得到python代码。

​ 放在Vscode中解码，发现出错了，原来是代码不完整，让ai补全，自己修正一下格式，得到flag。

1.2攻防世界-心仪的公司:

​ （1）. 这是一道流量分析题。根据题目提示，可以直接 http contains "shell" ,进行过滤。

​ 在后两个里追踪http流可以得到flag。

​ （2）.也可以在kali中写。用strings 命令查看，通过 { 找flag。

2.1总结：(1) 03 F3 0D 0A，是pyc文件头。(2).py 文件是Python源代码文件。包含了人类可读的Python代码。(3).pyc 文件是编译后的字节码文件，包含了Python虚拟机可以直接执行的二进制代码，对人类来说不可读。(4)把pyc文件转为py文件就行了。

2.2总结: 做流量分析时，可以打开kali用strings命令找flag。 strings 位置 | grep { ,或者grep "fl4g" -a 位置 ，fl4g是指与flag相似的等。
